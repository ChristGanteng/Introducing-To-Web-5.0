    • Lesson 1: Introduction to HTML

    In this lesson, you will learn the basics of HTML (Hypertext Markup Language), the markup language used to create the basic structure of web pages. HTML allows you to define elements such as headings, paragraphs, images, and links. This lesson will cover basic tags, attributes, and how to correctly structure HTML elements.

    Table of Contents

    1. What is HTML?

    HTML stands for HyperText Markup Language, a markup language used to create web pages. HTML works by marking up plain text using special elements called tags to form the structure of a web page.

    2. Basic Structure of HTML

    The HTML structure always starts with a declaration followed by the <html>, <head>, and <body> elements. Here is an example of a basic HTML structure:

    


    
    My First Web Page


    
    Welcome to My First Web Page
    
    
    This is my first paragraph.

    3. HTML Elements and Tags

    HTML elements are important parts of an HTML document. These elements start with an opening tag and end with a closing tag. Example:

    This is a paragraph element.
    

    This is a heading element.

    4. HTML Attributes

    HTML attributes are used to provide additional information about elements. For example, the style attribute is used to apply styles to text:

    This is a blue paragraph.

    5. Creating Paragraphs and Headings

    You can use the <p> element for paragraphs and <h1> to <h6> to create headings of different sizes. Example:

    Main Title (h1)
    

    Subsection Title (h2)
    

    This is an example paragraph.

    6. Adding Images and Links

    To add an image, use the <img> element with the src and alt attributes. For links, use the <a> element. Example:

    

    Click here to open another web page.
